### 自动装箱拆箱

装箱就是自动将基本数据类型转换为包装器类型  
拆箱就是自动将包装器类型转换为基本数据类型
```
Integer i = 10;  //装箱
int n = i;       //拆箱
```
int（4字节）	Integer
byte（1字节）	Byte
short（2字节）	Short
long（8字节）	Long
float（4字节）	Float
double（8字节）	Double
char（2字节）	Character
boolean（未定）	Boolean

#### 实现过程
装箱过程是通过调用包装器的valueOf方法实现的,而拆箱过程是通过调用包装器的xxxValue方法实现的.(xxx代表对应的基本数据类型)
