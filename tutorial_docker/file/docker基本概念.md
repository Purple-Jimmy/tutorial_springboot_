### docker基本概念

## 三大要素
* 镜像 image
镜像就是一个只读的模板,镜像可以创建容器,一个镜像可以创建很多容器
底层是unionFS(联合文件系统),实际由一层层的文件系统组成

* 容器 container
容器是用镜像创建的运行实例.可以创建一个或一组,可以启动,开始,停止,删除.每个容器都是相互隔离,保证安全的平台.


* 仓库 repository
仓库是集中存放镜像文件的场所.
仓库注册服务器(registry)上往往存放着多个仓库,每个仓库存放着多个镜像,每个镜像有不同的标签(tag)
