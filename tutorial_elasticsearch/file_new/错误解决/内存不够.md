## 锁住内存失败
```
ERROR: bootstrap checks failed
memory locking requested for elasticsearch process but memory is not locked
```
解决方案:
```
切换到root用户
编辑limits.conf配置文件
添加类似如下内容： 
sudo vim /etc/security/limits.conf



添加如下内容: 
* soft memlock unlimited 
* hard memlock unlimited 
备注：* 代表Linux所有用户名称


编辑 /etc/sysctl.conf
vm.swappiness=1
激活 sysctl -p
```
